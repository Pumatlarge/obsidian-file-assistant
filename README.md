# Obsidian File Assistant

An Obsidian plugin that provides a smart side panel to categorize and filter document content like headings, highlights, bold text, callouts, etc.

## How to Install

1. Download `main.js`, `manifest.json`, and `styles.css`.
2. Place them in your Obsidian vault's plugin folder: `.obsidian/plugins/obsidian-file-assistant/`.
3. Enable the plugin in Obsidian settings.

## Development

To modify and build the plugin:

1. Install dependencies: `npm install`
2. Build the plugin: `npm run build`
3. For development: `npm run dev`
